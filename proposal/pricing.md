# Pricing

**RFP #MC-2026-0417 — Meridian Components Inventory Dashboard Modernization**

We are proposing a **fixed-fee engagement** for the required scope (R1–R4), with desired items (D1–D3) priced separately as optional add-ons. Fixed-fee pricing gives Meridian cost certainty; the not-to-exceed figures below are our ceiling regardless of hours incurred.

---

## Base Engagement — Required Items (R1–R4)

| Item | Description | Fixed Fee |
|------|-------------|-----------|
| R1 | Reports module remediation — full defect audit and resolution | $18,000 |
| R2 | Restocking recommendations view — backend logic + frontend build | $32,000 |
| R3 | Automated browser test suite — all critical flows | $14,000 |
| R4 | Architecture documentation — current-state review and deliverable | $6,000 |
| | **Total — Base Engagement (8 weeks)** | **$70,000** |

---

## Optional Add-Ons — Desired Items (D1–D3)

| Item | Description | Estimated Fee |
|------|-------------|---------------|
| D2 | Internationalization — extend i18n to remaining modules | $12,000 |
| D3 | Dark mode — operator-selectable theme | $8,000 |
| D1 | UI modernization — visual refresh, scope TBD with stakeholders | $18,000–$28,000 |

D1 is range-priced because "current standards" is undefined in the RFP. We will provide a fixed fee for D1 once we have agreed on a design reference with Meridian stakeholders.

---

## Payment Schedule

| Milestone | Amount |
|-----------|--------|
| Contract execution | 25% ($17,500) |
| Phase 1 complete — R4 delivered, R1 remediated | 35% ($24,500) |
| Phase 2 complete — R2 live, R3 complete | 30% ($21,000) |
| Phase 3 complete — final acceptance | 10% ($7,000) |

---

## Assumptions

- Pricing assumes Meridian provides repository access and a working local development environment within 3 business days of contract execution.
- Travel is not included; all work is delivered remotely.
- Desired items (D1–D3) are billed separately against the optional add-on fees above; they are not included in the $70,000 base.
- If the Reports audit (R1) uncovers defects materially beyond the logged issue list and current codebase state, we will notify Meridian before proceeding and agree on scope in writing. This is not anticipated but is disclosed as a risk.
